# Mission Control: Token Refresh Tracker

A high-performance, dark-themed dashboard for tracking sensitive token refresh schedules. Built with premium glassmorphism aesthetics and mission-critical reliability.

![Mission Control Screenshot](timer_final_check_1775550665970.png)

## Features
- **Mission Control UI**: Sleek, immersive dark theme with neon accents and glassmorphism.
- **Micro-Interactions**: Real-time status badges, pulsing critical alerts, and haptic-style feedback.
- **Persistence**: Automated localStorage synchronization for session recovery.
- **Dual Alerts**: Browser notifications and audio signals when a timer expires.
- **Responsive Terminal**: Dynamic grid system optimized for both desktop and mobile command centers.

## Deployment to GitHub Pages
To deploy this project to your GitHub Pages:
1. Create a repository on GitHub.
2. Copy the remote URL (HTTPS or SSH).
3. Run the following in your terminal:
   ```bash
   git remote add origin YOUR_REMOTE_URL
   git branch -M main
   git push -u origin main
   ```
4. Navigate to **Settings > Pages** in your GitHub repo and select the `main` branch.
